# Trabajo Final – Big Data con Python

## Dataset
Se utiliza el Quality of Government Standard Dataset (Time-Series, January 2024), que contiene indicadores políticos, económicos y sociales para múltiples países a lo largo del tiempo.

## Región y países analizados
Este trabajo se centra en países del Magreb:
- Marruecos (MAR)
- Argelia (DZA)
- Túnez (TUN)
- Libia (LBY)
- Mauritania (MRT)

## Periodo de análisis
2000 – 2023

## Variables seleccionadas
- Democracia: vdem_polyarchy
- Corrupción: ti_cpi
- PIB per cápita: wdi_gdpc
- Desempleo: wdi_unemp
- Esperanza de vida: wdi_lifexp

## Pregunta de investigación
¿Cómo se relaciona el nivel de democracia con el desarrollo político y económico en los países del Magreb entre 2000 y 2023?
